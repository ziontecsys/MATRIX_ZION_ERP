ATENÇÃO — OPERAÇÃO DE ALTO RISCO
A importação de backup pode sobrescrever dados existentes no banco de dados. Registros com o mesmo ID serão atualizados com os dados do arquivo.
Faça um backup atualizado ANTES de importar. Erros nesta operação podem causar perda irreversível de dados.
